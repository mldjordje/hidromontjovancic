# Hidromont Jovancic (Next.js)

Repo sada sadrzi samo Next.js frontend za Vercel deployment.

## Lokalno

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Vercel env

- `NEXT_PUBLIC_SITE_URL=https://hidromontjovancic.rs`
- `NEXT_PUBLIC_API_BASE=https://api.hidromontjovancic.rs/api`

## Admin funkcije

- `/admin/projects` - dodavanje, izmena i brisanje projekata (tekst + slike)
- `/admin/orders` - pregled i obrada upita sa sajta

## Android instalacija admin-a

- Otvoriti `/admin` u Chrome na telefonu
- Kliknuti dugme "Instaliraj admin aplikaciju" (ili Chrome meni -> Add to Home screen)
